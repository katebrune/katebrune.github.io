# Typography

The `Typography` module exposes common typography elements.

## Components

| Element              | Description                |
| -------------------- | -------------------------- |
| Typography.Paragraph | A `paragraph` HTML element |
| Typography.Heading1  | A `h1` HTML element        |
| Typography.Heading2  | A `h2` HTML element        |
| Typography.Heading3  | A `h3` HTML element        |
| Typography.Code      | A `code` HTML element      |

## Attributes

### `color`

The font color to be applied to the typography element

| Options            |
| ------------------ |
| `'text-primary'`   |
| `'text-secondary'` |
| `'app-primary'`    |
| `'app-secondary'`  |
